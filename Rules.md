# Reglas de Programación - Proyecto Piscina
1. Las funciones en `components.js` deben ser PURAS (reciben datos, devuelven String HTML).
2. Prohibido usar `document.querySelector` o `appendChild` dentro de los componentes.
3. El estado de la aplicación se gestiona en `app.js`.
4. El renderizado se hace asignando el string al `innerHTML` del contenedor principal.
